# 🧘 EVEIL - Discipline & Ladder

**Eveil** est une application web (PWA) de suivi de discipline personnelle conçue pour transformer tes routines quotidiennes en une progression de type "jeu vidéo" (Ladder). L'objectif est double : maintenir une activité physique régulière et suivre ton temps d'abstinence (addictions).

---

## 🚀 Fonctionnalités

*   **Système de Rang (Ladder) :** Gagne des LP (Ladder Points) en complétant tes exercices. Grimpe de **Bronze** jusqu'à **Légende**.
*   **Exercices Personnalisables :** 10 slots d'exercices modifiables directement dans l'interface (clique sur le crayon ✏️).
*   **Système de Prestige :** Une fois le rang maximum atteint (1000 LP), passe au Prestige suivant pour débloquer des badges exclusifs.
*   **Suivi d'Abstinence :** Un timer précis suit ton temps de discipline. Ton "Arbre de vie" évolue avec le temps (🌱 → 🌿 → 🌳 → 🌸).
*   **Calendrier Dynamique :** Visualise tes performances passées. Les jours deviennent verts quand tes objectifs sont atteints.
*   **Sauvegarde Locale :** Tes données sont enregistrées dans ton navigateur. Tu peux aussi exporter/importer ton fichier de sauvegarde.

---

## ⚖️ Les Règles du Jeu (Économie des LP)

L'application est conçue pour récompenser la régularité et punir la négligence :

| Action | Impact LP |
| :--- | :--- |
| **Exercice terminé** | +1 LP |
| **Changement tout les 2 mois (Saison)** | -200 LP (Soft Reset) |

## 🛠️ Installation

L'application est contenue dans un seul fichier HTML, ce qui la rend extrêmement légère et privée.

1.  Allez sur https://paul59-dev.github.io/quiet/

### Sur Mobile (Mode App)
Pour une expérience optimale, tu peux l'installer comme une application :
* **Lien du site :** https://paul59-dev.github.io/quiet/
*   **iOS (Safari) :** Partager -> "Sur l'écran d'accueil".
*   **Android (Chrome) :** Menu (3 points) -> "Ajouter sur l'ecran d'accueil".

---

## 🛡️ Confidentialité
Aucune donnée n'est envoyée vers un serveur externe. Tout est stocké localement dans ton téléphone/ordinateur via le `localStorage`. Pense à utiliser le bouton **💾 Sauver** régulièrement pour garder une copie de ta progression.
